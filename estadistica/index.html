ESTADÍSTICA DESCRIPTIVA

RETURN
PROTOCOL_INFO // FLUX_DE_TREBALL

Els exercicis estan dissenyats per treballar amb les mateixes dades de forma progressiva. Pots generar dades noves o continuar amb les anteriors. Cada exercici construeix sobre l'anterior.

EXEC_01 // INTERVALS

CREACIÓ D'INTERVALS

Defineix els límits inferior i superior dels intervals. Calcula les marques de classe per organitzar les dades.

Dificultat: Bàsica // Temps: 10-15 min
 EXEC_02 // FREQÜÈNCIES

TAULES DE FREQÜÈNCIES

Construeix taules amb freqüències absolutes, relatives i acumulades per analitzar la distribució de dades.

Dificultat: Bàsica // Temps: 15-20 min
 EXEC_03 // CENTRALITZACIÓ

MESURES DE CENTRALITZACIÓ

Calcula mitjana aritmètica, mediana i moda. Identifica el centre de la distribució de dades.

Dificultat: Mitjana // Temps: 15-20 min
 EXEC_04 // DISPERSIÓ

MESURES DE DISPERSIÓ

Determina rang, variància i desviació estàndard. Analitza com es distribueixen les dades respecte la mitjana.

Dificultat: Mitjana // Temps: 20-25 min
 EXEC_05 // DISTRIBUCIÓ_NORMAL

CAMPANA DE GAUSS

Visualitza la distribució normal. Interpreta desviacions estàndard i percentatges de dades dins rangs específics.

Dificultat: Avançada // Temps: 20-30 min
